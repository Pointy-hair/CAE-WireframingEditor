HierachyTree

Source: HierachyTree.js:15

the hierachy tree visualize the hierachical structure of the ui elements in the wireframing editor and allows to edit the hierachy structure

Properties

The jsTree that handles and visualize the hierachy tree.

The html dom element as string.

the mxWindows-instance that contains the hierachy tree in the DOM.
